Button Component

The Button component is a versatile UI element used across our enterprise applications.

Design Guidelines

Our button design follows these principles:

  • Clear visual hierarchy through size and color
  • Distinct states (default, hover, active, disabled)
  • Consistent padding and spacing
  • Accessible color contrast ratios

Code Example


<button class="eds-button eds-button--primary">
    <span class="eds-button__text">Click Me</span>
</button>

Accessibility

Ensure buttons are accessible by following these guidelines:

  1. Use appropriate contrast ratios
  2. Maintain minimum touch target size of 44Ã—44px
  3. Include clear focus states
  4. Use semantic HTML elements
Note: Always test button components with screen readers to ensure they are properly announced.
